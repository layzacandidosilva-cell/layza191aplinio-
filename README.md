# layza191aplinio-# 🩰 O Mundo do Balé — Projeto Web

Bem-vindo ao repositório do projeto **O Mundo do Balé**! Este é um site responsivo e interativo desenvolvido para celebrar a arte da dança clássica e contemporânea, unindo design delicado, conteúdo informativo e interatividade para os usuários.

---

## 🎯 Objetivo do Projeto
O principal objetivo deste projeto é criar uma plataforma digital envolvente e educativa sobre o balé. O site busca desmistificar a dança, apresentar sua rica história, ilustrar os principais estilos e ensinar movimentos fundamentais de forma visual e interativa, servindo como uma porta de entrada para entusiastas e iniciantes.

---

## 🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web, sem a necessidade de frameworks externos, focando em performance e semântica limpa:

* **HTML5:** Estruturação semântica de todo o conteúdo (tags como `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
* **CSS3:** Estilização personalizada, paleta de cores em tons pastel (rosa e nude), tipografia fluida, design responsivo com Media Queries e animações suaves para transições.
* **JavaScript (ES6+):** Lógica de interatividade, incluindo navegação suave, filtros dinâmicos na tabela de movimentos, modo escuro/claro (*dark mode*) e um carrossel de imagens/depoimentos.

---

## ✨ Funcionalidades
* **Menu de Navegação Suave:** Âncoras que rolam a página suavemente até a seção desejada.
* **Dicionário de Movimentos Interativo:** Uma tabela com busca dinâmica via JavaScript, onde o usuário pode filtrar os passos de balé pelo nome.
* **Seletor de Estilos:** Abas ou cartões interativos que revelam detalhes sobre o Balé Clássico, Romântico e Contemporâneo ao clique.
* **Galeria Inspiradora:** Exibição visual de espetáculos e posições clássicas.
* **Design Totalmente Responsivo:** Perfeita visualização e usabilidade tanto em computadores quanto em tablets e smartphones.
